# IDE Integration

<PlanCompatibility features={["ai"]} />

Nutze mittwalds [DSGVO-konforme KI-Modelle](/docs/v2/platform/aihosting) direkt in deiner Entwicklungsumgebung fuer Code-Generierung, Completion, Refactoring und Debugging. Alle Modelle werden in Deutschland gehostet, ohne Nutzerdaten zu speichern oder zum Training zu verwenden.

## Verfuegbare Modelle

Der Service bietet mehrere Modelle fuer unterschiedliche Entwicklungsaufgaben. Eine vollstaendige Liste findest du in der [Dokumentation zu KI-Modellen](/docs/v2/platform/aihosting/models).

**Empfohlene Modelle:**
- **Ministral-3-14B-Instruct-2512**: Chat, Vision, General Assistants, multimodale UX, agentic coding, Tool Use
- **gpt-oss-120b**: Komplexe textzentrierte Aufgaben und praezise Automatisierungen

1. **API-Key holen**: In den [mStudio Profileinstellungen](https://studio.mittwald.de/profile/api-token)
2. **IDE waehlen**: Eine der Integrationen oben nutzen
3. **Endpoint konfigurieren**: `https://llm.aihosting.mittwald.de/v1`
4. **Modell waehlen**: `Ministral-3-14B-Instruct-2512`

## Best Practices

### Modellwahl

- **Ministral-3-14B-Instruct-2512**: Fuer die meisten Coding-Aufgaben, inkl. Tool Use und Vision
- **gpt-oss-120b**: Fuer komplexes Reasoning, praezise Dokumentation und anspruchsvolle Automatisierung

### Tipps fuer Long Context (256k)

- Nicht das ganze Repo senden, nur relevante Dateien/Funktionskoerper
- Vorherige Schritte zusammenfassen und IDs/Anker fuer Rehydration nutzen
- Antworten streamen, `max_tokens` nur bei Bedarf erhoehen
- Fuer Tools/Edits deterministisch bleiben: `temperature <= 0.2`
- Repo-Scope bewusst klein halten
- Bei Orchestrierung mehrstufiger Aenderungen bevorzugt strukturierte Outputs (JSON/Tool-Output) nutzen

### MCP Server Integration

Viele IDEs unterstuetzen [Model Context Protocol (MCP)](/docs/v2/agentic-coding/mcp/) Server fuer erweiterte Faehigkeiten:

- **Context7**: Aktuelle Library-Dokumentation
- **Brave Search**: Websuche mit Privacy-Fokus
- **PostgreSQL**: Datenbankabfragen und Schema-Inspektion
- **Directus**: Headless-CMS-Management
- **mittwald**: Infrastruktur-Management in natuerlicher Sprache

Details zur MCP-Konfiguration findest du in den jeweiligen IDE-Guides.
