# Cline mit mittwald

[Cline](https://cline.bot) ist ein KI-Coding-Assistent fuer Visual Studio Code, Cursor und Windsurf.

## Installation

Installiere die Cline-Erweiterung fuer deine IDE:

- [Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=saoudrizwan.claude-dev)
- Cursor (nutzt VS Code Marketplace)
- Windsurf (nutzt VS Code Marketplace)

## Konfiguration

Konfiguriere Cline ueber die UI:

1. Cline-Einstellungen oeffnen
2. **API Provider**: "OpenAI Compatible"
3. **Base URL**: `https://llm.aihosting.mittwald.de/v1`
4. **OpenAI Compatible API Key**: Dein mittwald LLM API-Key aus [mStudio](https://studio.mittwald.de/profile/api-token)
5. **Model ID**: `Ministral-3-14B-Instruct-2512`

## Features

Cline unterstuetzt:
- Autonomes Coding und Refactoring
- Datei-Bearbeitung und -Erstellung
- Terminal-Befehle ausfuehren
- Web-Browsing (bei passender Konfiguration)
- Tool Use und agentic Workflows
