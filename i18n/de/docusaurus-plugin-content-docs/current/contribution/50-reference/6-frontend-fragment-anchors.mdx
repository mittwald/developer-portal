---
title: Anchor für Frontend-Fragmente
---

Die Liste der verfügbaren Anchor für Frontend-Fragmente wird nicht mehr in dieser Dokumentation gepflegt.
Du findest die verfügbaren Anchor direkt im mStudio, wenn du Frontend-Fragmente für deine Extension konfigurierst.
Öffne dafür den Bereich **Entwicklung** deiner Extension und füge ein Frontend-Fragment hinzu oder bearbeite ein bestehendes Frontend-Fragment.
Die Konfiguration des Frontend-Fragments zeigt die verfügbaren Anchor und die Eigenschaften, die für den ausgewählten Anchor erforderlich sind.

Wenn du Anchor benötigst, die im mStudio nicht verfügbar sind, erstelle ein Issue im [Contributor Support GitHub-Repository](https://github.com/mittwald/contributor-support).

Jeder Anchor benötigt möglicherweise zusätzliche Eigenschaften, die beim Konfigurieren des Frontend-Fragmentes definiert werden müssen.
Derzeit sind die folgenden Eigenschaften für jeden Anchor erforderlich:

- **icon**: SVG-Icon für den Menüeintrag.
- **title**: Titel des Menüeintrags. Dies ist ein JSON-Objekt mit dem Sprachcode als Schlüssel und dem Titel als Wert.
  Zum Beispiel: `{"de": "Meine Extension"}`.
  Derzeit wird nur der Sprachcode "de" unterstützt.

Beispiel:

```json
{
  "/projects/project/menu/section/extensions/item": {
    "url": "https://my-extension.com/project/:projectId",
    "additionalProperties": {
      "icon": "<svg>...</svg>",
      "title": "{\"de\": \"Meine Extension\"}"
    }
  }
}
```
