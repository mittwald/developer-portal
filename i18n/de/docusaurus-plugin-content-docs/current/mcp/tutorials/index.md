# Tutorials

Tutorials sind geführte Lernpfade: Du gibst die Absicht vor, der Agent führt über MCP-Tools aus, und du verifizierst die Ergebnisse.

Jedes Tutorial in diesem Abschnitt folgt derselben Struktur:

1. Starter-Prompt zum Kopieren/Einfügen
2. Was der Agent automatisch macht
3. Was du noch tun musst
4. Wahrscheinliche Fragen und gute Antworten
5. Verifizierungs-Prompt
6. Rollback/Cleanup-Prompt
7. Wiederaufnahme-nach-Fehler-Prompt
