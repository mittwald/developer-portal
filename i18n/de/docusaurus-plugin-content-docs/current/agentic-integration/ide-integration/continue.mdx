# Continue mit mittwald

[Continue](https://www.continue.dev) ist ein Open-Source KI-Code-Assistent für JetBrains IDEs und Visual Studio Code.

## Installation

Installiere das Continue-Plugin:

- **JetBrains IDEs**: [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/22707-continue)
- **Visual Studio Code**: [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=Continue.continue)

## Konfiguration

Continue wird über eine `config.yaml` konfiguriert, typischerweise unter `~/.continue/`.

### Basis-Konfiguration

```yaml
name: mittwald
version: 1.0.0
schema: v1
models:
  - name: Ministral-3-14B-Instruct-2512
    provider: openai
    model: Ministral-3-14B-Instruct-2512
    apiBase: https://llm.aihosting.mittwald.de/v1
    apiKey: <INSERT-API-KEY>
    roles:
      - chat
      - edit
      - apply
    capabilities:
      - tool_use
context:
  - provider: code
  - provider: docs
  - provider: diff
  - provider: terminal
  - provider: problems
  - provider: folder
  - provider: codebase
```

Ersetze `<INSERT-API-KEY>` durch deinen mittwald LLM API-Key aus den [mStudio Profileinstellungen](https://studio.mittwald.de/profile/api-token).

## MCP Server Integration {#mcp}

Continue unterstützt MCP-Server. Ergänze in deiner `config.yaml` z. B.:

```yaml
mcpServers:
  context7:
    name: Context7
    command: npx
    args: ["-y", "@context7/mcp-server", "--stdio"]
    env: { CONTEXT7_API_KEY: "your-context7-api-key" }

  brave-search:
    name: Brave
    command: npx
    args: ["-y", "@modelcontextprotocol/server-brave-search", "--stdio"]
    env: { BRAVE_API_KEY: "your-brave-api-key" }

  postgres:
    name: PostgreSQL
    command: npx
    args: ["-y", "@modelcontextprotocol/server-postgres", "--stdio"]
    env:
      POSTGRES_CONNECTION_STRING: "postgresql://user:pass@localhost:5432/dbname"

  directus:
    name: Directus
    command: npx
    args: ["-y", "@directus/content-mcp", "--stdio"]
    env:
      DIRECTUS_URL: "https://your-directus-instance.com"
      DIRECTUS_TOKEN: "your-directus-token"

  mittwald:
    name: mittwald
    transport:
      type: sse
      url: https://mcp.mittwald.de/mcp
```

### MCP-Tools nutzen

Bitte den Assistenten explizit um Tool-Nutzung, z. B.:
- "Use Brave Search to find recent information about React 19..."
- "Qüry the PostgreSQL database to show all users..."
- "Use mittwald MCP to list my projects"

## Best Practices

### Sicherheit mit MCP-Servern

- Zugriff strikt begrenzen (Datenbanken, Domains, Pfade)
- Tool-Aufrufe für Audits protokollieren
- Für Tool-Planung niedrige Temperatur verwenden (`<= 0.2`)
- Diffs und Qüries vor Ausführung prüfen
- Wenn möglich read-only beginnen
