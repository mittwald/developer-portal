# IDE Integration

<PlanCompatibility features={["ai"]} />

Nutze mittwalds [DSGVO-konforme KI-Modelle](/docs/v2/platform/aihosting) direkt in deiner Entwicklungsumgebung für Code-Generierung, Vervollständigung, Refactoring und Debugging. Alle Modelle werden in Deutschland gehostet, ohne Nutzerdaten zu speichern oder zum Training zu verwenden.

## Verfügbare Modelle

Der Service bietet mehrere Modelle für unterschiedliche Entwicklungsaufgaben. Eine vollständige Liste findest du in der [Dokumentation zu KI-Modellen](/docs/v2/platform/aihosting/models).

**Empfohlene Modelle:**
- **Ministral-3-14B-Instruct-2512**: Chat, Vision, General Assistants, multimodale UX, agentic coding, Tool Use
- **gpt-oss-120b**: Komplexe textzentrierte Aufgaben und präzise Automatisierungen

## Schnellstart

1. **API-Key holen**: In den [mStudio Profileinstellungen](https://studio.mittwald.de/profile/api-token)
2. **IDE wählen**: Eine der Integrationen oben nutzen
3. **Endpoint konfigurieren**: `https://llm.aihosting.mittwald.de/v1`
4. **Modell wählen**: `Ministral-3-14B-Instruct-2512`

## Best Practices

### Modellwahl

- **Ministral-3-14B-Instruct-2512**: Für die meisten Coding-Aufgaben, inkl. Tool Use und Vision
- **gpt-oss-120b**: Für komplexes Reasoning, präzise Dokumentation und anspruchsvolle Automatisierung

### Tipps für Long Context (256k)

- Nicht das ganze Repo senden, nur relevante Dateien/Funktionskörper
- Vorherige Schritte zusammenfassen und IDs/Anker für Rehydration nutzen
- Antworten streamen, `max_tokens` nur bei Bedarf erhöhen
- Für Tools/Edits deterministisch bleiben: `temperature <= 0.2`
- Repo-Scope bewusst klein halten
- Bei Orchestrierung mehrstufiger Änderungen bevorzugt strukturierte Outputs (JSON/Tool-Output) nutzen

### MCP Server Integration

Viele IDEs unterstützen [Model Context Protocol (MCP)](/docs/v2/agentic-integration/mcp/) Server für erweiterte Fähigkeiten:

- **Context7**: Aktuelle Library-Dokumentation
- **Brave Search**: Websuche mit Privacy-Fokus
- **PostgreSQL**: Datenbankabfragen und Schema-Inspektion
- **Directus**: Headless-CMS-Management
- **mittwald**: Infrastruktur-Management in natürlicher Sprache

Details zur MCP-Konfiguration findest du in den jeweiligen IDE-Guides.
