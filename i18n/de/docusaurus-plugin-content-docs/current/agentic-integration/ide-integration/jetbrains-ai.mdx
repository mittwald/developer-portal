# JetBrains AI Assistant mit mittwald

Der JetBrains AI Assistant unterstützt OpenAI-kompatible Endpoints per Bring Your Own Key (BYOK). Das funktioniert mit IntelliJ IDEA, PHPStorm, WebStorm, PyCharm und weiteren JetBrains IDEs.

## Voraussetzungen

- JetBrains IDE (IntelliJ, PHPStorm, WebStorm, ...)
- mittwald API-Key aus den [mStudio Profileinstellungen](https://studio.mittwald.de/profile/api-token)

## Konfiguration

1. `Settings | Tools | AI Assistant | Models & API keys` öffnen
2. Unter "Third-party AI providers" **OpenAI-compatible** wählen
3. Konfiguration setzen:
   - **API Key**: Dein mittwald LLM API-Key
   - **Endpoint URL**: `https://llm.aihosting.mittwald.de/v1`
4. Verbindung testen
5. Optional: "Tool calling" aktivieren, wenn das Modell MCP-Tools nutzen soll

## Modellwahl

Wähle `Ministral-3-14B-Instruct-2512`. Das Modell unterstützt:
- Code-Generierung und Completion
- Chat und Dokumentation
- Vision-Fähigkeiten
- Tool Use und agentic Workflows

## Hinweis zum JetBrains AI Abo

Einige AI-Assistant-Funktionen können trotz Drittanbieter-Provider ein JetBrains AI Abo erfordern. Details findest du in der [JetBrains-Dokumentation](https://www.jetbrains.com/help/idea/ai-assistant.html).
