# Agentic Integration mit mittwald

<PlanCompatibility features={["ai"]} />

mittwald ermöglicht **agentic integration**: Du verbindest KI-Assistenten mit deinen Entwicklungs- und Betriebsabläufen, damit sie mehr als nur Code-Vorschläge liefern. Von Coding-Aufgaben bis zur Infrastruktur-Automatisierung führen Agenten mehrstufige Workflows aus, während du die Kontrolle behältst.

## Was ist Agentic Integration?

Klassische KI-Assistenten liefern oft nur Vorschläge und Vervollständigungen. **Agentic AI geht weiter**: Sie versteht dein Ziel, zerlegt es in Schritte, ruft die nötigen Tools auf und führt den Workflow über Code, Infrastruktur und Plattform-Operationen aus.

**Beispiel-Workflow:**
> "Deploye meine Express-App auf mittwald mit täglichen Backups"

Ein agentischer Assistent wird:
1. Ein Projekt in deinem mStudio-Account erstellen
2. Die Node.js-Laufzeit konfigurieren
3. Die Datenbank einrichten
4. Einen Backup-Plan anlegen
5. Deine Anwendung deployen
6. Die Fertigstellung inkl. URLs melden

Du beschreibst das Ergebnis, der Agent erledigt die Umsetzung.

## Drei Wege für Agentic Integration bei mittwald

### 1. IDE Integration

Nutze mittwalds [DSGVO-konforme KI-Modelle](/docs/v2/platform/aihosting) aus deutschem Hosting für Code-Generierung, Bearbeitung und Chat direkt in deiner Entwicklungsumgebung.

**Am besten für:** Code-Completion, Refactoring, Debugging, Test-Generierung

**Verfügbare Integrationen:**
- JetBrains IDEs (IntelliJ, PHPStorm, WebStorm)
- Visual Studio Code
- Cursor und Windsurf
- Zed
- CLI-Tools (OpenCode)

[Mehr zu IDE Integration →](./ide-integration/)

### 2. MCP (Model Context Protocol)

Verbinde KI-Assistenten über das Model Context Protocol mit der mittwald Infrastruktur-API. Verwalte Projekte, Datenbanken, Container, Domains und mehr in natürlicher Sprache.

**Am besten für:** Infrastruktur-Bereitstellung, Server-Management, operative Aufgaben

**Was du tun kannst:**
- Projekte erstellen und verwalten
- Container deployen und Datenbanken konfigurieren
- Domains und SSL-Zertifikate verwalten
- Backups und Cronjobs überwachen
- Logs und Metriken abrufen

[Mehr zu MCP →](./mcp/)

### 3. Agent Skills

Vorgefertigte Workflow-Hilfen für komplexe Aufgaben wie Migrationen und Zero-Config-Deployments. Skills liefern strukturierte Schritt-für-Schritt-Unterstützung ohne individuelle Mega-Prompts.

**Am besten für:** Komplexe Migrationen, Deployment-Workflows, Plattform-Onboarding

**Verfügbare Skills:**
- **mittwald-migrate**: Konversationelle Migration von jeder Plattform zu mStudio
- **mittwald-zerodeploy**: Zero-Config-Deployment mit Auto-Erkennung

[Mehr zu Agent Skills →](./agent-skills/)

## Erste Schritte

### Schnellstart: IDE Integration

1. Hole dir deinen API-Key in [mStudio](https://studio.mittwald.de/profile/api-token)
2. Wähle deine [IDE-Integration](./ide-integration/)
3. Konfiguriere den Endpoint: `https://llm.aihosting.mittwald.de/v1`
4. Wähle das Modell: `Ministral-3-14B-Instruct-2512`

### Schnellstart: MCP

1. Installiere einen [MCP-kompatiblen Client](./mcp/getting-connected/)
2. Verbinde dich mit: `https://mcp.mittwald.de/mcp`
3. Authentifiziere dich über OAuth oder API-Token
4. Starte Infrastruktur-Management in natürlicher Sprache

### Schnellstart: Agent Skills

1. Installiere die [mittwald Agent Skills](./agent-skills/) für deinen Assistenten
2. Starte mit Phrasen wie "migrate to mittwald" oder "deploy to mittwald"
3. Folge dem konversationellen Workflow

## Datenschutz und Compliance

Alle KI-Services von mittwald sind:
- **DSGVO-konform** und in Deutschland gehostet
- **Ohne Datenspeicherung** zu Trainingszwecken
- **Sicher** durch OAuth 2.1 mit PKCE
- **Granular steuerbar** durch Scopes und Berechtigungen

## Nutzungsgrenzen

Nutzungsgrenzen richten sich nach deinem mStudio-Tarif. Prüfe deine [Plan-Details](https://studio.mittwald.de/account/limits) für aktuelle Limits.
