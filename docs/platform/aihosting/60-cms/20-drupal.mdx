## Using the mittwald AI provider

- https://www.drupal.org/project/ai_provider_mittwald

### Installation

Installation:

```
composer require drupal/ai_provider_mittwald
drush en ai_provider_mittwald
```

### Configuration

1. Go to **Configuration** -> **System** -> **Keys**, click on **Add key** and insert your mittwald AI Hosting API key.
1. Go to **Configuration** -> **AI** -> **Provider Settings**, click on **mittwald Authentication**; in the field **mittwald API key**, select the key you created in the previous step and save the configuration.
1. Go to **Configuration** -> **AI** -> **AI default settings**, select **mittwald** as default provider for
   1. Translate text
   1. Chat (and Chat with Image Visiion, and chat with Tool Calling)
   1. Embeddings
