## Picking models {#picking-models}

- Start with `Ministral-3-14B-Instruct-2512` for broad, scalable, cost-conscious chat and entry multimodal (text+image) workflows.
- Upgrade to `Devstral-Small-2-24B-Instruct-2512` for demanding, nuanced, business-critical tasks where top answer quality, agentic workflows, and image understanding are vital.
- Use `gpt-oss-120b` for large text-centric workloads, advanced automations, or when large document context is required.
- Choose `Qwen3-Embedding-8B` for all use cases involving search, recommendation, clustering, or knowledge graph building.
- Deploy `whisper-large-v3-turbo` for any audio transcription or voice-command needs.

:::tip
For optimal ROI, start with the smallest model that meets your quality bar—then upgrade *only* where you need additional depth or reliability!
:::
