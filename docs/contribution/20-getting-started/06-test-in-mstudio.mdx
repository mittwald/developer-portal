## Create an Extension Instance

To test the Extension locally, you must integrate it into mStudio through an Extension Instance.

For the Reference Extension, the following prerequisites from previous steps must be fulfilled:

- A project has been created
- The database is running
- The Extension is running locally
- Context, permissions, Frontend Fragment URL, and webhook URL are configured
- The tunnel is running

Now inside the "Leitfaden Extensionverwaltung" (Extension management guide) on tab **"Ãœbersicht"**, you can install the Extension in your own organization
or in a project of your organization.

:::info Server or project plan
If you also want to run the Extension in mStudio as described in the [Deployment step](../deploy),
you should not choose a project plan. Instead, book a server and create the project there to be able to use the container hosting.
:::
