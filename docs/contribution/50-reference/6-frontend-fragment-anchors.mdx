---
title: Frontend fragment anchors
---

The list of available frontend fragment anchors is no longer maintained in this documentation.
You can find the available anchors directly in mStudio when configuring frontend fragments for your Extension.
Open the **Development** section of your Extension and add or edit a frontend fragment.
The frontend fragment configuration shows the available anchors and the properties that are required for the selected anchor.

If you need an anchor that is not available in mStudio, please create an issue in the [Contributor Support GitHub Repository](https://github.com/mittwald/contributor-support).

Each anchor may need additional properties to be defined while configuring the frontend fragment. Currently, the following properties are required for each anchor:

- **icon**: SVG icon for the menu entry.
- **title**: Title of the menu entry. This is a JSON object with the language code as key and the title as value.
  For example: `{"de": "Meine Extension"}`. Currently, only the language code "de" is supported.

Example:

```json
{
  "/projects/project/menu/section/extensions/item": {
    "url": "https://my-extension.com/project/:projectId",
    "additionalProperties": {
      "icon": "<svg>...</svg>",
      "title": "{\"de\": \"Meine Extension\"}"
    }
  }
}
```
